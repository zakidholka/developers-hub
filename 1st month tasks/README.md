## Task 1 — Iris Dataset Exploration & Visualization

### Objective
Analyze and visualize the Iris dataset to understand feature distributions, relationships, and data trends.

### Dataset
Iris Dataset (loaded using Seaborn)

### Concepts Covered
- Data Loading
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Analysis

### Visualizations
- Scatter Plot
- Histograms
- Box Plots
- Pair Plot

### Libraries Used
- Pandas
- Seaborn
- Matplotlib

### Key Insights
- Iris-setosa is clearly separable from other species.
- Petal-related features provide better classification separation.
- Some outliers were observed in sepal width.
